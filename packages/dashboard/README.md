# Dashboard

This is a vue application for registrar client in TFGridV4.

## Prerequisites

- node 20.10.0 or higher

## Installation

1. **Clone the Repository**

   ```bash
   git clone https://github.com/threefoldtech/tfgridv4-sdk-ts
   cd tfgridv4-sdk-ts
   ```

2. **Build the Registrar Client Package**

   Navigate to the `registrar_client` package directory and build the package:

   ```bash
   cd packages/registrar_client
   yarn install
   yarn build
   ```

3. **Navigate to the Dashboard Package Directory**

   ```bash
   cd ../dashboard
   ```

4. **Install Dependencies**

   ```bash
   yarn install
   ```

5. **Build the Application**

   ```bash
   yarn build
   ```

6. **Run the Application**

   ```bash
   yarn dev
   ```
